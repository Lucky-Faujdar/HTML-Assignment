1. Head-Tag:

    This is the head section of the HTML document, which contains meta-information about the document, such as the title of the web page that appears in the browser's title bar or tab

  2. Title-Tag:

    This is the title tag, which sets the title of the web page. In this example, it's set to "Hello, World!".

  3. body-Tag:

    This is the body section of the HTML document, which contains the visible content of the web page, such as headings, paragraphs, and more.

  4. h1-Tag:

    This is a heading tag, used to define a top-level heading. In this example, it displays the text "Hello, World!" as a heading

  5. P-Tag:

    This is a paragraph tag, used to define a paragraph of text. In this example, it displays the text "This is my first HTML page." as a paragraph.

  6. ul-Tag:

    It stands for an unordered list, which means every list item will have a bullet point in front of it.

  7. ol-Tag:

    It stands for ordered list, which means every list item will have a number in front of it.

  8. dl-Tag:

    It stands for Definition List. It represents a list of terms and their corresponding definitions

    The smallest container in the table is a table cell, which we can create by using the tag which denotes table data.

  13. Tr-Tag:

    This element defines a row of cells in a table

  14. Th-Tag:

    This tag defines a header cell in an HTML table. The text in elements are bold and centered by default

  15. "rowspan":

    This attribute allows a cell to occupy and cover multiple rows, making it taller than a single row which means spanning a cell against several rows. To use rowspan, we need to include this attribute within or of the cell you want to span multiple rows. The value of rowspan specifies how many rows the cell should span.

  16. "colspan":

    This allows a cell to span across multiple columns, making it wider than a single column which means spanning a cell against several columns. To use colspan, we need to include this attribute within or of the cell you want to span multiple columns. The value of colspan specifies how many columns the cell should span.

  17. "Cellpadding":

    creates a gap between content and cell borders, making the table look tidier and easier to read.

  18. “cellspacing”:

    This attribute specifies the amount of space we want between the cells of the table
